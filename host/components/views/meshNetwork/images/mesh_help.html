Quick Reference Guide

The Basic UI can be switched between a virtual office and a smart home environment. Mesh nodes are located in the virtual environment and take a role based on their specific location. For example, a mesh node in the office image location next to the fire extinguisher labelled doorbell will enable the magnetic contact and make a beep sound when the provided magnet is removed.

When a mesh node is powered, it will provision and show up on one of the locations in the Node Bar above the image with its color and mesh Unique Address (uAddr). See image below for the pink node with mesh uAddr 2. The Green Strata mesh node is the mesh hub node that is always present which Strata uses to send messages into the BLE mesh network.

Except for the provisioner node at the Strata location (uAddr 1), nodes can be moved to a different location in the virtual office by left-clicking and dragging the node to any other open node on the Node Bar. The small circle in the image will change to the color of the node and also show the corresponding uAddr. The node will have different behavior and responses depending on its location.

Virtual Office Environment Node Locations

 

Security Camera

When left-clicking the small circle in the image all mesh nodes will change their color to the color of the mesh node at this location. If there is no mesh node at the location all of the mesh node lights will be switched off.

Doorbell

The mesh node will enable the magnetic contact to detect any magnet underneath it. If there is no magnet present a beep will sound for 2 seconds. Both the connection and disconnection events are sent from the mesh nodes to the Strata hub and can be seen on the console log. When left-clicking on the small circle located in the image, the Strata node will send a beep event to the mesh node and it will beep also for 2 seconds.

Door

The mesh node will enable the magnetic contact to detect any magnet underneath it. If there is no magnet present the light alarm will activate for 10 seconds where all of the mesh nodes and the virtual office will blink red. The red flashing will continue until the magnet is detected again or the node is moved to a different visual location.

Dimmer

The mesh node will enable the magnetic contact to detect magnetic field angle. When moving a node to this location make sure it is not close to the magnet as it will calibrate for 1 second before first use. After calibration the node can be rotated slowly over the magnet and it will dim the lights on all mesh nodes. See the full Mesh Platform User Guide for more information.

Strata Hub (Provisioner)

The mesh node at this location cannot be moved. When left-clicking on the node in the image it will toggle between RF power levels: white light indicates lowered RF power and green light indicates nominal RF power. Lower RF power level can be helpful in the Relay demonstration. See Relay below for more information.

Robotic Arm

When left-clicking the small circle in the image all mesh nodes will change their color to the color of the mesh node at this location. If there is no mesh node at the location, all mesh nodes light will be switched off.

Solar Panel / Relay

When left-clicking on the node in the image it will toggle between: Relay on and turning its light white and Relay off. This is used in the Relay demonstration. With the Relay enabled, nodes normally outside the Provisioners range can communicate by relaying information through the Solar Panel / Relay node.

HVAC

This mesh node can be physically located out of range of all other nodes except of the Solar Panel node. When enabling Relay on the solar Panel node this node will respond to commands from the nodes that are out of range, but when relay is disabled it will not. When left-clicking the small circle in the image all mesh nodes will change their color to the color of the mesh node at this location. If there is no mesh node at the location, all mesh nodes light will be switched off.

Smart Home Environment Node Locations

Window

When this location is clicked, the electronic frosting on the glass window toggles and reveals the city skyline outside.

Strata Hub (Provisioner)

The mesh node at this location cannot be moved. When left-clicking on the node in the image it will toggle between RF power levels: white light indicates lowered RF power and green light indicates nominal RF power.

Door

The mesh node will arm to detect any magnet underneath it. If there is no magnet present the light alarm will activate for 10 seconds where all of the mesh nodes and the virtual smart home lights will blink red. The red flashing will continue until the magnet is detected again or the node is moved to a different visual location.

Lights

Clicking on this location will cycle the lights on all the nodes and in the virtual smart home through several different colors.
